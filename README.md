# Chess ♟️

A modern, lightweight web chess experience with customizable opponents, board themes, and player options. Play instantly in your browser with a clean interface and smooth gameplay.

📱 **Play Online on Vercel:** [Live Demo](https://chess-zeta-opal.vercel.app/)

## 🚀 Introduction

**Chess Game** brings the timeless strategy of chess to your browser in a simple yet engaging format. With selectable AI opponents, board themes, and player sides, it’s designed for both casual players and enthusiasts.

### Key Features
* 🤖 **AI Opponents:** Challenge different skill levels for varied gameplay.
* 🎨 **Board Themes:** Personalize your match with unique visual styles.
* ⚪⚫ **Play As White or Black:** Choose your side and start strategizing.
* 🖥️ **Clean Interface:** Minimalist design focused on smooth play.

## 🕹️ How It Works

The game runs directly in your browser:

1. **Setup Screen:** Select your opponent, board theme, and playing color.
2. **Gameplay:** Make moves turn by turn with clear controls.
3. **Match Flow:** Play until checkmate, resignation, or draw.

## 🛠️ How to Setup & Run Locally

This project uses only three files and requires no external dependencies.

### Prerequisites
A modern web browser (Chrome, Firefox, Edge, or Safari).

### Local Installation Steps
1. **Clone or Download the Repository.**
2. Ensure the following files are in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in your browser.
4. Start playing!
