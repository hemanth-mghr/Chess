# Chess Game 🎮♟️

## Introduction
This is a fun and interactive chess game designed for players of all levels. It offers customizable opponents, board themes, and player options, making each match unique and engaging.

👉 **Play Online:** [chess-hemanth15.vercel.app](https://chess-hemanth15.vercel.app)

## How It Works
- Choose your opponent from different AI levels.
- Select a board theme to personalize your experience.
- Pick your side (White or Black) and start the match.
- Play turn by turn with clear controls and an intuitive interface.

## How to Install
To run the game locally:
1. Download or clone this repository.
2. Ensure the following files are in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in any modern web browser.
4. Start playing!

## Conclusion
This project is a simple yet enjoyable way to play chess in your browser. It’s lightweight, customizable, and perfect for casual play or practice. Contributions and feedback are welcome!
